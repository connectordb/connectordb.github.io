Set Up ConnectorDB

For detailed instructions, please visit the installation instructions.

tl;dr

{% highlight bash %} apt-get install postgresql redis-server tar -zxvf connectordb0.3.0.tar.gz cd connectordb ./connectordb create mydatabase --user=myuser:mypassword ./connectordb start mydatabase ./connectordb run mydatabase {% endhighlight %} And ConnectorDB is running on port 8000. But you should really go through the detailed instructions, since you miss a lot of setup with this tl;dr.
