Set Up ConnectorDB

For detailed info, please visit the installation instructions. If you have trouble, feel free to go on gitter for help.

tl;dr

{% highlight bash %} sudo apt-get install postgresql redis-server # ConnectorDB manages databases itself sudo systemctl disable postgresql.service sudo systemctl stop postgresql.service sudo systemctl disable redis-server.service sudo systemctl stop redis-server.service # Get the files wget https://github.com/connectordb/connectordb/releases/download/v0.3.0a1/connectordb0.3.0a1.tar.gz tar -zxvf connectordb0.3.0a1.tar.gz cd bin ./connectordb create mydatabase --user=myuser:mypassword ./connectordb start mydatabase ./connectordb run mydatabase {% endhighlight %}

And ConnectorDB is running on localhost port 8000. But you should really go through the detailed instructions, since you miss a lot of setup with this tl;dr.
