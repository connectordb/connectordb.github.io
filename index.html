Analyze Your Data

Use ConnectorDB's powerful tools for data analysis.

PipeScript

Perform data analysis using PipeScript, a tiny analysis language built entirely for time series data. Do aggregations and transformations on your data, all in a single line of code.

While it is an unfamiliar little language, the tutorial should get you up to speed in no time at all. You can also play around with it below. On the left is input data, and on the right is your script's result. A list of available transforms is available here.

Sample Scripts

  • Get the weekday for each datapoint
  • Find average number of steps taken per day
  • Get only the datapoints where you were walking
  • Get the total number of datapoints
  • Get both mean and total number of steps
  • Return the sum of steps taken every other datapoint.

Datasets

Combine multiple streams of data into a tabular dataset using PipeScript's built-in interpolators. No need for complicated JOIN statements.
